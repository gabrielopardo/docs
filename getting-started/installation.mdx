# Installation

Arc Manager is a Windows desktop application. No additional software is required.

## Downloading the Application

1. Download the latest version from the official Arc Manager website
2. You will receive either:
   - **Installer (.exe)** — Installs Arc Manager to your computer
   - **Portable (.exe)** — Runs directly without installation

## Running for the First Time

1. Double-click the downloaded file
2. You may see a **Windows SmartScreen** warning:
   - Click **More info**
   - Click **Run anyway**
3. The application will open and prompt you for your license key

## Activating Your License

1. Enter the license key from your purchase receipt
2. Click **Activate License**
3. The main application will load

## Where Your Data Is Stored

Arc Manager stores all data locally on your computer in your user folder.  
See **Backup & Data Safety** for details.
